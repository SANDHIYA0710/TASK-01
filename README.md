Task 1 – Print Formatting
This task explains how to control the **format of printed output** using the `sep` parameter in the `print()` function. The `sep` argument decides what should come **between multiple values** when they are printed. In the first print statement, space is used as the separator, so the words appear normally. In the second print statement, `" | "` is used, so the items are separated by a vertical bar. This is useful when displaying data in a **structured or readable format**.

Task 2 – Variables
This task demonstrates how to **store data in variables**. Here, `name`, `age`, and `city` store different types of values like string and integer. These variables are then printed in a single line using `-` as a separator. Variables are important because they allow us to **store, reuse, and manipulate data** in a program.

Task 3 – Multiple Assignment
This task shows how to assign **multiple values to multiple variables in one line**. Instead of writing separate assignment statements, Python allows assigning values together like `name, age, student = ...`. This makes the code **shorter, cleaner, and easier to read**. The Boolean value `True` represents a logical state such as yes/no or active/inactive.

Task 4 – Indexing
This task explains how to **access specific characters in a string** using indexing. In Python, indexing starts from `0`, so the first letter is at position `0`, the third letter at position `2`, and the last letter can be accessed using negative indexing like `-1`. Indexing is very useful for **string processing, validation, and data extraction**.

Task 5 – Arithmetic Operators
This task demonstrates basic **mathematical operations** in Python. Operators like `+`, `-`, `*`, and `/` perform addition, subtraction, multiplication, and division. The modulus operator `%` gives the remainder, `**` calculates power, and `//` performs floor division which removes the decimal part. These operators are widely used in **calculations, analytics, and logical computations**.

Task 6 – BODMAS Expression
This task shows how Python follows **operator precedence rules (BODMAS)** while solving expressions. Python first calculates exponentiation (`**`), then multiplication (`*`), and finally addition (`+`). This helps programmers understand **how complex expressions are evaluated step-by-step** without confusion.

Task 7 – Assignment Operators
This task explains **shortcut assignment operators** like `+=` and `/=`. These operators update the value of a variable by performing an operation and assigning the result in a single step. For example, `num += 25` means add 25 to the existing value of `num`. This makes programs **more efficient and reduces the number of lines of code**.

Task 8 – Comparison Operators
This task demonstrates how to **compare two values** using operators like `>`, `<`, `==`, `!=`, `>=`, and `<=`. The result of these comparisons is always either `True` or `False`. These operators are essential in programming because they are used in **decision-making statements such as if conditions, loops, and validations**.

 Task 9 – String Comparison

This task explains how Python compares two strings using the equality operator ==. In this example, "apple" and "Apple" are compared. Even though the words look similar, Python returns False because it is case-sensitive, meaning uppercase and lowercase letters are treated as different characters. String comparison is commonly used in login systems, form validation, and data matching.

Task 10 – Logical Operators
This task demonstrates the use of logical operators such as and, or, and not to combine or modify conditions. The and operator returns True only if both conditions are True. The or operator returns True if at least one condition is True. The not operator reverses the result of a condition (True becomes False and vice versa). Logical operators are important for decision making, filtering conditions, and controlling program flow.

Task 11 – Membership Operator
This task shows how to check whether a particular value exists inside a list using membership operators in and not in. If the value is present in the list, Python returns True; otherwise it returns False. This concept is useful in search operations, data validation, and checking availability of items in collections.

Task 12 – Swap Variables
This task demonstrates how to exchange the values of two variables using a single-line statement a, b = b, a. Python allows swapping without using a temporary variable, which makes the code simpler and more efficient. Swapping is a common operation used in sorting algorithms, data processing, and interview coding questions.

Task 13 – Bitwise XOR
This task explains the bitwise XOR (^) operator, which performs operations at the binary level. Python converts numbers into binary form and compares each bit. XOR returns 1 only when the bits are different and 0 when they are the same. This operation is used in encryption techniques, error detection, networking, and low-level system programming.
